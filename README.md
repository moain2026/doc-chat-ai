# DocChat AI

  An AI Document Chat application built with React + Vite + TypeScript.

  ## Features
  - 🔐 Authentication (login/signup)
  - 📁 Document upload (PDF, DOCX, XLSX, TXT)
  - 💬 AI chat with streaming responses
  - 🌙 Dark mode
  - 📱 Responsive design
